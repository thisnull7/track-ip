## ☠️ TENTANG

**TRACK IP** adalah terminal-based IP tracker dengan antarmuka brutal dan informatif. Dirancang untuk memberikan informasi geolokasi akurat dari sebuah alamat IP secara instan. Dilengkapi dengan tampilan terminal yang menyeramkan, input interaktif, dan tautan langsung ke **Google Maps** untuk melihat lokasi target.

**Dibangun dengan presisi. Digunakan tanpa ampun.**

---

## 🎯 FITUR UTAMA

| Fitur | Deskripsi |
|-------|-----------|
| 🌍 **Geolokasi Akurat** | Negara, kota, koordinat, ISP, dan banyak lagi |
| 🗺️ **Google Maps Pin** | Tautan langsung ke lokasi target di Google Maps |
| 🎨 **Tampilan Brutal** | UI terminal gelap dengan aksen merah dan tengkorak |
| ⌨️ **Input Interaktif** | Masukkan IP langsung atau via argumen CLI |
| 🔍 **Deteksi Proxy/VPN** | Mendeteksi apakah IP menggunakan proxy, VPN, atau hosting |
| ⚡ **Ringan & Cepat** | Hanya menggunakan requests dan colorama |

## 🛠️ INSTALASI

### Prasyarat
- **Python 3.7+** terinstal di sistem
- **Git** untuk cloning repositori

### Langkah Instalasi

```bash
# Clone repositori
git clone https://github.com/thisnull7/track-ip.git

# Masuk ke direktori
cd track-ip

# Install dependensi
pip install -r requirements.txt

Run
python ip.py
